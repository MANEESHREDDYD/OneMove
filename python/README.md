# OneMove Intelligence Package

This package contains deterministic ML scoring, analytics, and data quality modules for the OneMove Marketplace.

## Setup Instructions

### Windows
```bash
cd python
python -m venv .venv
.venv\Scripts\python -m pip install -e .[dev]
```

### macOS / Linux
```bash
cd python
python3 -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
```
