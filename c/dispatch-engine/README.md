# OneMove C Dispatch Engine

This is an optional portfolio subsystem demonstrating systems-level performance optimization.
It calculates Haversine distances and ranks nearest-partner candidates.

## Purpose
- High-performance dispatch candidate ranking.
- Fast heuristic calculations in native C.

## Running Locally
This module is not required for the core Next.js application to run.
If you have a C compiler (`gcc` or `clang`) and `make` installed:

```bash
make
make test
make benchmark
```
