🚀 **Just wrapped up a major portfolio project: OneMove!** 🚀

OneMove isn't just another UI mockup. It's a full-stack, four-sided super-app marketplace built entirely from the ground up using **Next.js** and **Supabase**. 

I wanted to demonstrate portfolio-ready engineering depth by focusing heavily on the backend architecture and data orchestration. 

**Key Technical Highlights:**
🔒 **Secure Multi-Tenant Backend:** Architected a polymorphic database schema protected by deep PostgreSQL Row Level Security (RLS), ensuring strict data isolation between Customers, Merchants, Partners, and Admins.
📊 **Data Engineering:** Built a custom Metric Store with automated cron-based data pipelines that aggregate raw transactional events into daily analytics snapshots.
🧠 **Deterministic ML/AI Scoring:** Implemented an internal intelligence layer running rule-based demand forecasting, dispatch optimization, and risk modeling scripts.
🧪 **A/B Experiments & MLOps:** Built an internal experimentation platform to simulate synthetic traffic, calculate variant conversions, and log all execution states into an MLOps pipeline for full observability.
🛡️ **Rigorous QA:** Validated the entire application lifecycle and security boundaries using Playwright End-to-End tests and Vitest.

If you're interested in full-stack architecture, data engineering pipelines, or MLOps, check out the project!
