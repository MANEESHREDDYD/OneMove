## OneMove - Marketplace Intelligence Platform

- **Built a four-sided marketplace MVP** using Next.js, TypeScript, Supabase PostgreSQL, and RLS, supporting customer, merchant, partner, and admin workflows across rides, food, grocery, and courier services.
- **Architected a secure multi-tenant backend** by implementing deep PostgreSQL Row Level Security (RLS) policies, ensuring strict data isolation for over 150+ demo users across 4 distinct role contexts.
- **Engineered automated data pipelines** to aggregate 500+ synthetic raw transactional events into a unified Metric Store, enabling real-time analytical dashboards and operational insights.
- **Developed a deterministic ML/AI intelligence layer** capable of generating 1,500+ risk, dispatch, and demand forecast records via rule-based scoring algorithms without relying on paid third-party LLM APIs.
- **Designed an internal A/B experimentation engine** utilizing synthetic traffic simulation to calculate directional conversion and revenue-per-user metrics across feature variants.
- **Implemented an MLOps logging framework** to track and audit the execution duration, status, and generated row counts of all automated intelligence and pipeline scripts.
- **Established a rigorous QA testing matrix** by authoring 34 comprehensive Playwright End-to-End and Security tests to validate critical transaction paths and RLS boundary enforcement.
